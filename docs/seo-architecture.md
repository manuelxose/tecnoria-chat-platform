## Oportunidades futuras de contenido

- Comparativas:
  - chatbot para web vs centro de ayuda conversacional
  - IA conversacional para SaaS vs chatbot de soporte clásico

- Landings evergreen:
  - chatbot IA para SaaS B2B
  - chatbot IA para soporte al cliente
  - asistente IA para documentación técnica
  - widget IA para captación de leads

- Contenido táctico:
  - cómo integrar un chatbot IA en una web corporativa
  - cómo construir una base de conocimiento útil para IA conversacional
  - errores al desplegar asistentes IA en producto
  - cómo medir conversaciones, leads y preguntas no resueltas

## Gaps semánticos detectados antes del rework

- baja diferenciación entre funcionalidades, integraciones y casos de uso
- poco vocabulario útil alrededor de captación, onboarding y soporte
- señal demasiado débil para keywords comerciales de “plataforma”, “integración” y “casos de uso”
- blog sin estrategia visible cuando no había artículos
- FAQ insuficiente para absorber dudas previas a demo

## Defaults asumidos

- El sitio seguirá siendo bilingüe con paridad estructural ES/EN.
- La conversión principal seguirá siendo demo consultiva.
- No se usarán testimonios, logos ni cifras inventadas hasta disponer de prueba publicable.
